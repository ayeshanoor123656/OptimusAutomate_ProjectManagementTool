# TaskFlow – Project Management Tool

## Overview

TaskFlow is a collaborative project management application that allows teams to organize work using boards and tasks. Users can create boards, assign tasks to team members, track progress using drag-and-drop functionality, add comments, and manage deadlines.

## Features

* User Registration and Login
* Project Boards
* Task Creation and Deletion
* Task Assignment
* Due Dates
* Comments on Tasks
* Drag-and-Drop Task Management
* Dashboard Statistics
* My Tasks View
* Team Collaboration

## Tech Stack

### Frontend

* React.js
* Vite
* Axios
* React Router DOM
* DnD Kit

### Backend

* FastAPI
* Python

### Database

* MongoDB Atlas

### Deployment

* Frontend: Vercel
* Backend: Render

## Installation

### Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
